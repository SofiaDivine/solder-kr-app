# Маршрут пайки — Solder Route Optimizer

Програмний продукт для розв'язання задачі складання оптимального маршруту
роботизованої паяльної головки з урахуванням ресурсного обмеження (запас припою).

**Автори:** Танчук С.Ю., Міхрін Е.О., Моріна А.В.  
**Дисципліна:** Дослідження операцій в ІУС  
**Університет:** НТУУ «КПІ ім. Ігоря Сікорського»

---

## Постановка задачі

Паяльна головка має відвідати **n** точок пайки на платі, починаючи і
закінчуючи в базі **R(0, 0)**. Місткість подавача припою обмежена **Q**
одиницями. При нестачі припою головка повертається на базу для дозаправки
(час дозаправки — **τ** с). Мета — мінімізувати сумарний час маршруту.

## Алгоритми

### Жадібний алгоритм — `solve_greedy`
- Евристика найближчого сусіда з перевіркою ресурсного обмеження
- Складність: **O(n²)**
- Базовий детермінований розв'язок для порівняння

### Мурашиний алгоритм — `solve_aco`
- ACO з імовірнісним вибором переходів (τ^α · η^β), η = 1/c_ij
- Дозаправка в R при нестачі припою (як у табл. 2.2)
- Параметри за замовчуванням: **m=20**, **N_iter=100**, **α=1**, **β=2**, **ρ=0.1**
- Складність: **O(N_iter · m · n²)**

Покращення ACO відносно жадібного:

**δ = (Z_greedy − Z_ACO) / Z_greedy × 100%**

## Приклад з коду

```python
from model.problem import SolderPoint, ProblemInstance
from model.algorithms import solve_greedy, solve_aco, AcoParams

task = ProblemInstance(
    points=[SolderPoint(25, 40, 35), SolderPoint(55, 15, 45)],
    Q=100, tau=8, v=15,
)
task.build_dist()

greedy = solve_greedy(task)
aco = solve_aco(task, AcoParams(m=10, n_iter=50, seed=42))

delta = (greedy.total_time - aco.total_time) / greedy.total_time * 100
print(f"Жадібний: {greedy.total_time:.2f} с")
print(f"ACO:      {aco.total_time:.2f} с")
print(f"δ:        {delta:.2f}%")
```
